# AI-агенты

Поддержка AI-кодинг агентов в сгенерированных проектах.

## Содержание раздела

- [CLAUDE.md / LLMS.md](llms-md.md) — документ-инструкция для AI-агентов

## Обзор

Go Project Starter может генерировать документ-инструкцию, который помогает AI-кодинг агентам (Claude Code, Codex, Cursor и др.) понимать архитектуру и правила проекта без дополнительных вопросов.

### Включение

```yaml
main:
  name: myproject
  generate_llms_md: true
```

### Выбор имени файла

Генератор автоматически выбирает имя файла:

| Состояние проекта | Генерируемый файл | Причина |
|---|---|---|
| Нет `CLAUDE.md` | `CLAUDE.md` | Claude Code автоматически читает этот файл |
| Есть свой `CLAUDE.md` | `LLMS.md` | Чтобы не перезаписать файл пользователя |

### Что содержит документ

- Описание архитектуры проекта (трёхслойная архитектура)
- Правила работы с генерируемым кодом (disclaimer-маркеры, `_gen` файлы)
- Команды сборки, тестирования, генерации
- Описание транспортов, воркеров, драйверов
- Структура каталогов проекта
